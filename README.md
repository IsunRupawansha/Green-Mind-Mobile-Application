GreenMind AI
Intelligent Herbal Plant Identification and Ayurvedic Knowledge Retrieval System

GreenMind AI is an AI-powered mobile application designed to support Ayurvedic doctors, researchers, students, and practitioners by providing accurate and easy access to Sri Lankan native herbal plant information.

The system combines:

1.Herbal Plant Identification using YOLOv8
2.AI-powered RAG Chatbot
3.Disease Remedy Follow-up System
4.Cross-platform Mobile Application

 Features
1. AI Herbal Knowledge Chatbot (RAG Agent)
Retrieval-Augmented Generation (RAG) based chatbot
Answers herbal and medicinal plant questions
Uses semantic search with vector embeddings
Reduces hallucinations by retrieving real herbal data
Technologies
OpenAI Embeddings
FAISS Vector Database
LangChain
LangGraph
FastAPI

3. Herbal Plant Image Classification
Upload plant image from mobile app
Detects and identifies medicinal plants
Provides plant information and medicinal usage

Add remedies and treatments
